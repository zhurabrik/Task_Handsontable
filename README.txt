Для запуска необходимо, чтобы на ПК был установлен Node.js

1. Находясь в основной папке проекта, выполнить команду: npm i
2. Дождаться установки пакетов
3. Выполнить команду: npm start
4. Открыть index.html и нажать на кнопку. В терминале выведется, какие id были отфильтрованны

В исходной таблице 4 строки, третья строка пустая. 
Мы фильтруя исключаем пустые строки, соответсвенно должны вывестись id: [ 1, 2, 4 ] 
(первой строке соответствует id = 1, и далее по порядку)
